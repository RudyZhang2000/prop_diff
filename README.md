# Player Prop Comparison Script

## Overview
This script fetches player prop lines for all sports from PrizePicks and Underdog, compares them, and prints the differences.

## Requirements
- Python 3.7+
- requests

## Install dependencies
```bash
pip install requests
```

## Run the script
```bash
python compare_props.py
```

## Output
Each difference is printed as a dictionary with player, sport, prop type, and the lines from each source.
